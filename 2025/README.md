# Advent of Code 2025

This year's solutions are built in C++17.

## Requirements

* g++
* make

## Build and Run a solution

```sh
make <DAY>
```

## Build a Solution

```sh
make build<DAY>
```

## Run a Solution

```sh
make run<DAY>
```

## Generate a New Solution Directory

The following command infers the next day based on the maximum day in the [2025](/2025/) directory.

```sh
./generate.sh
```
