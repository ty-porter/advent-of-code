# Advent of Code 2025

This year's solutions are built in C++17.

## Requirements

* g++
* make

## Generate a New Solution Directory

```sh
make new
```

## Build Solution Runner

```sh
make build
```

## Run all Solutions with Benchmarking

```sh
make run
```

## Run a Single Solution with Benchmarking

```sh
make run DAY=<DAY>
```
